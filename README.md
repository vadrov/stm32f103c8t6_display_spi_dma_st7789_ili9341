# STM32F103C8T6_display_spi_dma_st7789_ili9341
 STM32F103x display driver. Supports displays with controllers st7789, ili9341 and compatible with them, working on the spi interface. DMA support.

Connection:

LCD_SCL ---> PA5
LCD_SDA ---> PA7
LCD_DC  ---> PA2
LCD_RES ---> PA3
LCD_CS  ---> PA4
LCD_BLK ---> PA6
